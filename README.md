# SCOS Login Automation (Playwright)

## Overview
This project contains Playwright automation test cases for the SCOS React (web) login module.

## Features Covered
- Authentication (valid/invalid login)
- Institute & Role-based flows
- Validation scenarios
- UI behavior (loader, button state, toggle)
- Performance scenarios (multiple clicks, loader)
- Security scenarios (invalid inputs)

## Tech Stack
- Playwright
- JavaScript
